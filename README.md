Objective: Segment customers into VIP, Loyal, At-Risk, and Lost categories using Recency, Frequency, and Monetary (RFM) analysis.

500K+ transaction records

Common Table Expressions (CTEs)
Window Functions (NTILE)
CASE WHEN
DATEDIFF
STR_TO_DATE
Aggregations

Project Workflow:

Raw Data → Data Cleaning → Order Aggregation → RFM Metrics → Customer Scoring → Customer Segmentation → Business Insights

Business Insights Generated:

VIP customer identification
Customer loyalty analysis
Churn-risk detection
Revenue contribution by segment
